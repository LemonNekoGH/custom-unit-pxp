# Custom Unit PXP

A set of packages to create a custom unit `pxp` that is a combination of `px` and `var`, it's very useful in game development.

## Packages

- [@lemonneko/postcss-plugin-pxp](packages/postcss-plugin-pxp/README.md): A postcss plugin to transform static css declarations with `pxp` unit to `calc` expression.
- [@lemonneko/vue-plugin-pxp](packages/vue-plugin-pxp/README.md): A Vue plugin to introduce the `v-style-pxp` directive to bind inline styles with `pxp` unit and transform it.
- [@lemonneko/vue-compiler-plugin-pxp](packages/vue-compiler-plugin-pxp/README.md): A Vue compiler plugin to transform `pxp` unit to `calc` unit.
